Reading 01 - Tips on Creating Stylized 3D Content

Techiniques and tools:

- Practiced on Polycount, making personal work and portfolio

- Learn through new software packages, new workflows, and practice

- Low poly art: hardware is less powerful and content designed to be scalable to operate on many platforms

- Low poly thinking: use real- time solution as opposed to renders

- Sketchfab for personal work, web-based 3D model viewer which can be accessed on variety of platforms and hardware, but need low poly thinking

- Dioramas: iterative process, start off as grey meshes, then chiseled and detailed. Constanting moving, adding, taking away, and make sure there is a consistent level of detail

- View at all angles so make sure there are interesting elements 360

- All polymodelling done in 3DS Max, only start using other packages like ZBrush if there are organic complex textures

- Iterate textures and lighting/mood in tandem to modelling, lay down large areas of colour then work on detailing much later. Larger -> small is important for a balanced piece with the same level of detail across the whole environment.

- Tip: having a nice colour palette is imporant and should be decided earlier on

- Use squint test to evaluate if the scene is heading in the right direction in development

- Composition is hard: keep contrast between horizontal and vertical surfaces, use colours from a small band (unless want it to stand out), separate colour groups to separate buildings, maintain breathing space (especially around objects with high focus)

- Scale: real-world scale is irrelevant, but it is important to maintain consistent scale. Keep models of humans dotted around for reference

- Keep mesh density and texel density, atlas map textures. Texture alpha is devil for performance

- Texturing: isolate small clusters of props in 3DS Max, then projection bake into final atlas UVs

- Process: place down flat colour using brush with granular texture, add soft highlight for edges, add softer cavity dirt or occlusion, add extra detais with painterly brush, add adjustment layers to props

- Last step is outlining entire scene
