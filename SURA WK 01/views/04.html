Reading 04 - Please, it's super cool to use Three.js to make 2D images look 3D

Three.js:

- Convert 2D to 3D using Three.js + React technology stack, converts 2D cartoon pictures into 3D visual effects.

- Image has a visual depth of field effect of 3D

Steps: prepare a picture as the original material, best to divide picture content into multiple levels to achieve the 3D depth of field effect. Open the picture in Photoshop , create several layers according to the number of layers needed, copy map to each layer, then edit each layer according to the division of the depth of the layer scene, and use the magic of Stick Tool and Lasso Tool Remove the excess, then export each layer individually as footage.

- OrbitControls used for lens track control, TWEEN used for lens tween animation
